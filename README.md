# Cinema Ticket System

A Java-based cinema ticket booking system that manages movie screenings, seat reservations, and ticket purchases.

## Features

- **Movie Management**: Store and display movie information including title, genre, duration, and showtime
- **Cinema Hall**: Manage cinema halls with customizable seating arrangements
- **Seat Reservation**: Book and reserve seats with real-time availability tracking
- **Ticket System**: Generate tickets with customer details, seat information, and pricing

## Project Structure

- `Main.java` - Entry point of the application
- `Movie.java` - Movie class with details like title, genre, and duration
- `CinemaHall.java` - Cinema hall management and seat layout
- `Seat.java` - Individual seat representation with booking status
- `Ticket.java` - Ticket generation and customer information
- `CinemaSystem.java` - Core system logic for managing bookings

## Getting Started

1. Clone the repository
2. Open the project in your Java IDE
3. Run `Main.java` to start the application

## Requirements

- Java 8 or higher
